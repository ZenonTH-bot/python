### 2. Trình bày cách máy tính thực thi lệnh.
Quá trình thực thi lệnh bao gồm các bước sau:

  * **IF (Instruction Fetch):** Nhận lệnh 
  * **ID (Instruction Decode):** Giải mã lệnh 
  * **DF (Data):** Nhận dữ liệu 
  * **EX (Execution):** Thực hiện
  * **DS (Data Store):** Lưu trữ kết quả

### 3\. Trình biên dịch và trình thông dịch là gì? Lập bảng phân loại các ngôn ngữ lập trình.

**Định nghĩa:**

  * **Trình biên dịch:** Là một chương trình máy tính thực hiện việc chuyển đổi toàn bộ chương trình ngôn ngữ cấp cao (chương trình nguồn) sang chương trình ngôn ngữ máy.
  * **Trình thông dịch:** Thay vì chuyển đổi toàn bộ chương trình nguồn như trình biên dịch, trình thông dịch chỉ chuyên đổi một mệnh đề của chương trình và thực hiện đoạn mã kết quả ngay, sau đó nó tiếp tục chuyển đổi mệnh đề thứ 2 rồi thi hành đoạn mã kết quả thứ 2 và cứ thế đến hết.

**Bảng phân loại ngôn ngữ lập trình:** 

| Ngôn ngữ lập trình | Trình biên dịch | Trình thông dịch |

| **Python** | | X |
| **C và C++** | X | |
| **PHP** | | X |
| **Ruby** | | X |

### 4. Viết chương trình in ra màn hình dòng chữ: “Hello world\! Tôi là xxx”.

*(Trong đó xxx là tên sinh viên)*

**Tên sinh viên:** Nguyễn Sỹ Hoàng Anh 

**Cách 1:** Sử dụng biến

```python
my_string = "Hello world! Tôi là Nguyễn Sỹ Hoàng Anh"
print(my_string)
```

**Cách 2:** In trực tiếp 

```python
print("Hello world! Tôi là Nguyễn Sỹ Hoàng Anh")
```
